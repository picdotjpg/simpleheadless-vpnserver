How to use: (For Windows)

Download the config file

Extract the contents

Download Openvpn (if you havent already)

Set the .ovpn File you just downloaded as the config

Connect to the Server

How to Use: (for Linux)

Download the .ovpn Config file

Use your systems package manager to download openvpn

Note, for *nix systems that do not have openvpn in their repositories, you can build from source from Openvpn Github

After openvpn and its dependancies are installed, use the cli tool to connect to the vpn server with the key you already downloaded

If everything was set up right, you should see this message

How to setup a server

Download the .sh & run it (reqiures super user access)

wget https://raw.githubusercontent.com/picdotjpg/simpleheadless-vpnserver/master/setupvpn.sh && sudo chmod +x setupvpn.sh && sudo ./setupvpn.sh
